Base plate

The plate follows the outline of the text itself, extended outward by this margin - not a fixed rectangle/circle. Base thickness (2mm) and text height (1mm) are fixed.

Keyring hole

Position relative to the default (left edge, vertically centered). Matches the offset convention used in popular MakerWorld parametric keychains. Hole size is fixed at 3mm.

Filament colors

Base and text are separate solids so each can be assigned its own AMS filament color, like the reference model.
